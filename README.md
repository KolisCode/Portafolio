🔗 **Ver en vivo:** *(agregar URL de deploy)*

## Stack

- React 18
- Vite 6
- CSS por componente

## Desarrollo local

```bash
npm install
npm run dev    # → http://localhost:5173
```

## Build

```bash
npm run build    # genera dist/
```

## Proyectos incluidos

- **LotesRB** — portal inmobiliario (Angular 21 + NestJS + PostgreSQL)
- **Biodont** — sistema odontológico (Angular + Node.js + SQLite)
